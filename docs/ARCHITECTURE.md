# معمارية Systematize Framework

## التعريف

المنظومة هي إطار حوكمة وتشغيل للمشروعات البرمجية متعددة المراحل، وليست حقيبة أدوات متناثرة ولا مجرد واجهة أوامر.

## الطبقات

| الطبقة | المسؤولية | الحالة المستهدفة |
| --- | --- | --- |
| Governance DSL | توجيه الوكيل والحوكمة الإجرائية | أوامر موصوفة ومصنفة |
| Runtime Engine | تنفيذ الأفعال الفعلية | محرك Node وحيد |
| Compatibility Shell | التوافق مع بيئات PowerShell | أغلفة تستدعي Node |
| Control Plane | التكوين والذاكرة والحالة | عقود أوضح وتوليد أدق |
| Policy Layer | السياسات والمعايير الثقيلة | منفصلة عن الأوامر داخل `docs/policies` |
| Extension Layer | التكاملات الاختيارية | لا تدخل القلب |

## حدود القرار التنفيذي

- القرار التنفيذي الرسمي يجب أن يمر عبر:
  `.Systematize/scripts/node`
- لا يجوز أن تملك طبقة PowerShell منطق أعمال مستقلًا عن المحرك.
- لا يجوز اعتبار أوامر الحوكمة نفسها أوامر تشغيل بالضرورة.
