# طبقة السياسات الحاكمة

> هذه وثيقة مرجعية.
>
> للبداية التشغيلية:
>
> ```text
> docs/START_HERE.md
> ```

يحتوي هذا المجلد على السياسات الحاكمة الثقيلة التي فُصلت عن سطح الأوامر حتى يبقى السطح الأول أخف وأوضح.

## الغرض

- إبقاء ملفات الأوامر مركزة على التنسيق والمسار.
- نقل القواعد الثقيلة والقابلة لإعادة الاستخدام إلى طبقة سياسات مستقلة.
- جعل الحد بين سطح الحوكمة وطبقة السياسات حدًا صريحًا يمكن الرجوع إليه بسهولة.

## السياسات المستخرجة

```text
docs/policies/systematize-policy.md
docs/policies/research-policy.md
docs/policies/tasks-policy.md
docs/policies/analyze-policy.md
docs/policies/checklist-policy.md
docs/policies/implement-policy.md
docs/policies/clarify-policy.md
```

## القاعدة التشغيلية

كل أمر حاكم يجب أن يشير صراحة إلى سياسة مرجعية واحدة على الأقل، وأن يعاملها بوصفها المصدر المعياري للنصوص الثقيلة ومعايير التقييم وعقد المخرجات.
